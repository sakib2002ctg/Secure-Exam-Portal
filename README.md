# 🛡️ ExamGuard - Secure Online Exam Platform

A front-end web application designed to simulate a secure, proctored online testing environment. Built with a modern, dark-themed UI, this platform includes built-in JavaScript security measures to maintain academic integrity during remote examinations.

## ✨ Features
* **Modern UI/UX:** A professional, responsive dark-mode interface designed for focus and readability.
* **Security & Proctoring Scripts:**
  * Disables right-click context menus.
  * Blocks keyboard shortcuts for Copy, Cut, and Paste.
  * Detects and logs when a user switches browser tabs or minimizes the window.
* **Active Exam Dashboard:**
  * Live countdown timer.
  * Interactive question palette to track "Answered", "Unanswered", and "Marked for Review" questions.
* **Multi-Page Navigation:** Includes seamlessly linked pages for Student Login, Exam Guidelines, Technical Support, and Team Credits.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure and multi-page routing.
* **CSS3:** Custom styling, flexbox/grid layouts, hover effects, and linear gradients.

* **Font Awesome:** Integrated vector icons for a polished look.

## 🚀 How to Run Locally
Because this is a static website, no servers or databases are required to view the front-end!
1. Clone or download this repository to your local machine.
2. Extract the folder if downloaded as a `.zip`.
3. Double-click the `index.html` file to open it in your default web browser.
